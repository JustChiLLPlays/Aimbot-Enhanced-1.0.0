## 🧠 Mod Functionality

> When Player Open Pause Menu, On The game, Then On The Right Top Corner , There Will Be A Button Called "AimOptimizer_Enhanced" , Which Will Open The AimOptimizer_Enhanced Menu.
> In The AimOptimizer_Enhanced Menu, There Will Be Some Options Like "Enable/Disable" ,  "Close Menu" , "Show Help" , Normal Aim , Enhanced Aim , About Mod (Show A Dialog Menu Where Clear That This Mod Is A True Unfair Cheat) , "About Mod"   etc.

After Player Clicks "Enable" , The Aim Of The Player To The Entity Will Be Optimized And Enhanced, That Results In A More Precise Aim.It Will Also Help The Player To Hit The Entity More Easily.Entity Means Player, Mob, Mob Entity, Player Entity, etc. The Aim of The Player Is Little Bit Locked, When The Player Is Aiming At The Entity.
This Should Help The Player To Aim More Precisely, And To Hit The Entity More Easily, Like An Aimassist or Aimbot.

After Player Clicks "Disable" , The Aim Of The Player To The Entity Will Be Reset To The Original One.

Add These Commands Also:
- `/aimoptimizer enable` : Enable The AimOptimizer_Enhanced Features.
- `/aimoptimizer disable` : Disable The AimOptimizer_Enhanced Feature.
-  /aimoptimizer help : Show The Help Message, That How To use All Features.
- `/aimoptimizer about` : Show The About Mod Message, That Clear That This Mod Is A True Unfair Cheat.
-  /aimoptimizer Developer : Show The Developer Message, That Show The Name Of The Developer, That Created This Mod.
